<u>Installation of Anaconda and Jupyter Notebook</u>  
Visit official Anaconda Website and download *Anaconda Individual Edition*.  
Choose *Python 3.xx.xxx* suitable version for OS.  
Double click on downloaded file to run application.  
* Run the setup
* Agree to License Agreement
* Select individual installation type
* Choose appropriate destination folder. Preferably in C: Drive
* Enable to add Anaconda to my PATH and register as default.
* Installation will complete.
